## C. 매니저 대시보드 (메인, 한 페이지) ⭐ 핵심

**매니저 시점**: 로그인 → 한 화면에 모든 게 보임 → 5초 안에 "내 호텔이 어떤 상태인지" 파악

**헌법 7조(5초 파악) 매니저 적용 — 7영역 한 페이지 (스크롤만):**

| 영역 | 표시 내용 | 데이터 출처 |
|---|---|---|
| ① 진행 단계 시각화 | 입력 → 영상 제작 → 노출 → 예약 시작 | hotels.status |
| ② 호텔 정보 | 호텔명, 위치, Agoda 매칭 상태 | hotels |
| ③ 콘텐츠 | 제작 영상 수, YouTube 링크 | videos |
| ④ 노출 채널 | 8/8 채널 노출 현황 | videos.channel_id |
| ⑤ 예약 결과 | 실예약 건수, 6개월 누적 | bookings |
| ⑥ 6개월 보장 + 재제작 | 보장 만료일, 재제작 권리 | hotels.guarantee_until |
| ⑦ 매출 추정 | Agoda 커미션 추정 | bookings × commission |

**중요 — 통찰 5(조회수 딜레마):**
- ✅ 메인: 채널 노출 / 콘텐츠 결과 / 예약 / 매출 추정
- 🟡 보조: 조회수 (참고용 작은 표시)
- ⭐ 차별화: 6개월 보장 + 재제작 권리 **항상 노출**
