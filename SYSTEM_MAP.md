## 3. 자동으로 도는 것 — 크론 (서버 자동 · 사이트 닫아도 돔)

> 각 봇은 **정해진 시각에 잠깐 켜져 몇 분 일하고 꺼진다.** 전부 서버가 알아서 함. 시간은 **한국시간(KST)**.
> 🔴 **새 봇을 켜거나 끄면 이 표를 반드시 고친다.**

| 봇 | 언제(KST) | 하는 일 | 비용 | 문제 시 볼 곳 |
|---|---|---|---|---|
| **kw-survey** | **매시간** | 키워드 자동 순환 — 예약 많은 미조사 도시 발굴+측정(유튜브 자동완성·트렌드·경쟁). 선택 새치기·품질 게이트·건너뛰기(survey_skip) | 무료 | `?dry_run=1`(다음 도시)·survey_skip·snapshot.status |
| **yt-views** | **매시간** | 발행 영상 조회수 — 나이별(첫날 3h/12h/24h·1일~1달 주1·1달+ 월1·D-059) | 거의 0 | publications.next_fetch_at·view_count |

**코드만 있고 지금 안 도는 것**(vercel.json 크론에 없음): `invoice-expire`·`invoice-retention`·`manager-campaign`·`receipt-overdue` — 필요하면 vercel.json에 추가해야 켜짐.

**데이터 흐름 (D-069 파이프라인 · 아고다 1차·구글 최소):**
```
아고다 월업뎃 → 지역채우기(아고다주소·매일·무료) → 폐업체크(사라진곳만 구글·월1)
             → 키워드 순환(예약순·유튜브·매시간·무료) → 조회수(나이별·매시간) → 화면(캐시값)
```

**문제 빨리 찾기 (증상 → 어디):**
- 키워드 안 채워짐 → kw-survey `?dry_run=1`·survey_skip(건너뛴 도시)
- 완성인데 지도 0 → 그 나라 파서 규칙 없음(D-070)·아고다 재고 미로드(0% 도시)
- 조회수 안 옴 → yt-views·next_fetch_at·status=published+video_id
- 같은 호텔 두 개 → 확인함 "📍좌표 같은 호텔" 합치기(D-071)
- 폐업 오판 → hotel-closed-check는 아고다 담는 8도시만·구글 확인 후만

**관련 결정:** D-059(조회수)·D-069(파이프라인)·D-070(나라별 지역+측정)·D-071(호텔 병합).

## 4. 비용 — 유료는 구글 최소뿐

- **유료 = 구글 좌표뿐.** 이번 달 634건 / 무료 5,000건 = **실제 0원**.
- 아고다 파일로 좌표 **82%를 구글 0건으로** 채움 → 좌표 봇 **7/21경 끔 → 유료 0**.
- 조사(유튜브 무료 1만/일) · 아고다 파일(무료) · 구글 트렌드(무료) = **전부 0원**.
- 고정비 = Vercel Pro 구독 · Supabase.
- 🔴 **구글 필드마스크 절대 변경 금지** — Pro 무료(5,000)가 Enterprise(1,000 유료)로 추락.

## 5. 안전장치 — 꼬임 방지 5 (위험도 순)

1. 💰 **유료 한도** — 구글 필드마스크 금지. 이번 달 634/4,500 캡.
2. 🔒 **백업 안전** — 깃허브 **100MB 상한**. 넘으면 그날 백업 통째 실패 = 되돌리기 불능. **아고다 도시 담기 전 반드시 계산**.
3. ⏰ **새벽 시간표** — 크론 겹치면 구글 한도 초과 → 가입 흐름까지 막힘. 구글 한도는 **태평양 자정 리셋**. 조사 봇은 좌표 봇 끈 뒤 붙임.
4. 🔑 **토큰·권한** — 저장 창구 commit 30·query 60/시간. 깃허브 열쇠(`GITHUB_PAT`) **2026-08-25 만료** → 미리 갱신. 워크플로(`.github/workflows/*`)는 클로드 수정 불가(403).
5. 📋 **운영 규칙** — 스튜디오엔 **수수료 표시 금지**. DB 쓰기(UPDATE·DELETE) 전 백업 먼저. 매니저 호텔(`user_id` 있음)은 **읽기만**.
