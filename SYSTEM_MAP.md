## 4. 비용 — 유료는 하나뿐, 그것도 곧 0

- **유료 = 구글 좌표뿐.** 이번 달 634건 / 무료 5,000건 = **실제 0원**.
- 아고다 파일로 좌표 **82%를 구글 0건으로** 채움 → 좌표 봇 **7/21경 끔 → 유료 0**.
- 조사(유튜브 무료 1만/일) · 아고다 파일(무료) · 구글 트렌드(무료) = **전부 0원**.
- 고정비 = Vercel Pro 구독 · Supabase.
- 🔴 **구글 필드마스크 절대 변경 금지** — Pro 무료(5,000)가 Enterprise(1,000 유료)로 추락.

## 5. 안전장치 — 꼬임 방지 5 (위험도 순)

1. 💰 **유료 한도** — 구글 필드마스크 금지. 이번 달 634/4,500 캡.
2. 🔒 **백업 안전** — 깃허브 **100MB 상한**. 넘으면 그날 백업 통째 실패 = 되돌리기 불능. **아고다 도시 담기 전 반드시 계산**.
3. ⏰ **새벽 시간표** — 크론 겹치면 구글 한도 초과 → 가입 흐름까지 막힘. 구글 한도는 **태평양 자정 리셋**. 조사 봇은 좌표 봇 끈 뒤 붙임.
4. 🔑 **토큰·권한** — 저장 창구 commit 30·query 60/시간. 깃허브 열쇠(`GITHUB_PAT`) **2026-08-25 만료** → 미리 갱신. 워크플로(`.github/workflows/*`)는 클로드 수정 불가(403).
5. 📋 **운영 규칙** — 스튜디오엔 **수수료 표시 금지**. DB 쓰기(UPDATE·DELETE) 전 백업 먼저. 매니저 호텔(`user_id` 있음)은 **읽기만**.
