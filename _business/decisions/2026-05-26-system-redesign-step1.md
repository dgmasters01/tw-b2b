## D1. 자동 저장 창구 신설

**한 줄:** 파일을 GitHub에 올릴 때 매번 대표님 거치지 말고, Claude가 자동 창구로 곧장 박는다.

- **창구 주소:** `gohotelwinners.com/api/ops/github-commit`
- **보내는 것:** 어디(path), 무슨 내용(content), 무슨 이유(message)
- **돌아오는 것:** "200 = 성공", 영구 보관 주소(commit URL)
- **한도:** 한 시간에 30번 (충분)
- **결과:** 대표님께 "이거 GitHub에 올릴까요" 질문이 사라짐

## D7. 정석 자가 검증 5기준

**한 줄:** 대표님께 선택지 보내기 전, 5개 통과한 것만.

| 기준 | 통과 조건 |
|---|---|
| **장기 안정** | 6개월 후에도 작동하는가 |
| **표준 패턴** | 업계에서 다 쓰는 방식인가 |
| **유지보수** | 나중에 고칠 때 쉬운가 |
| **단일 진실** | 같은 정보가 한 군데에만 있는가 |
| **롤백 안전** | 잘못되면 되돌릴 수 있는가 |

- **1개라도 NO = 폐기.** 정석만 비교 가능
- **임시 우회·수동 작업·하드코딩 = 자동 탈락** → 대표님께 보이지도 않음
- **결과:** 대표님은 "정석 중에 어느 게 좋을까"만 고르면 됨
