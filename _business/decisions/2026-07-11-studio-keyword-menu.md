# 2026-07-11 스튜디오 키워드 메뉴 (UX/UI) — 분석·추천 엔진 + 수요 트렌드 인텔리전스

> 스튜디오 6메뉴 중 다섯 번째 "키워드". 라이브 코드(_content/youtube/키워드-실측.md v1·keywords/{도시}.csv·youtube.js §4) 검증 기반.

## 목적
키워드 메뉴 = **"다음에 뭘 만들지"를 데이터로 뽑아주는 분석·추천 엔진** + **수요 트렌드 인텔리전스(선행 지표)**. 키워드 나열이 목적이 아니라, 실측(수요·경쟁)으로 발굴하고 월별로 누적해 계절·전년 트렌드를 만들어 "어느 나라·도시·위치·성급 호텔이 언제 필요한지 미리" 알려주는 자리. **사람은 확인·미세조정만**(수동 채택 없음).

## 구조 (메인 + 3단계 드릴)
### 메인(홈) = 국가 개요
- **월 선택**: 상단 월 스트립(최근·앞 달 빠른 접근) + **연·월 선택기**(데이터 쌓일수록 과거 조회·여러 해 비교). 월 의미 2축 = **지난 달=실측 복기 / 앞 달=겨냥 준비**(기본 동작).
- **국가 개요 순위**: 이번 달 어느 나라에 집중할지(착수 우선도=수요 상승+미개척 기회). 나라별 신호·미개척 수·추천 콘텐츠 수.
- **선행 준비**(여기가 제자리 — 여러 나라 가로지름): 착수=피크−3달(리드타임 3달), 4개월 앞까지. 겨냥 월별 콘텐츠·나라·도시·상태(늦음/지금 착수/여유). → 미리 만들어 예약 건다.

### 도시 개요 (나라 클릭)
그 나라 도시별(오사카·도쿄·후쿠오카…): 이번 달 수요·미개척·추천 수·월별 추이. 도시 클릭 → 상세.

### 도시 상세 (도시 클릭)
- **① 콘텐츠 추천**: 순위 + 근거 + **넣을 키워드 세트**(자수 포함). [이걸로 만들기].
- **② 키워드 순위 자동 정리**: 기회점수순, 월별 추이 스파크라인, **보는 용**(수동 채택 없음). 채택 후보 상위 ~30(500자 범위) + [더 보기]. "기준 월" 명시.
- **③ 자동 채택 태그란**: 규칙 적용 500자 자동 완성. 미세조정만. [복사]·[§4 저장].
- **④ 미개척 기회**: 순위 + 키워드. **클릭하면 펼쳐져 키워드별 노출량(월간 수요)** 표시 → 얼마나 큰 기회인지 판단.
- **⑤ 그 도시 수요 트렌드**: 월별, **여러 해(2024·2025·2026) 겹쳐** 계절 패턴 확인.
- **연관 키워드**: 자동완성 파생·호텔명 축약형 별도 섹션(누르면 채택 추가).

### 네비게이션
상단 경로(breadcrumb) 클릭으로 상위 단계 이동("일본›오사카"에서 일본=도시개요, 이번 달 개요=메인). 홈으로 언제든 복귀.

## 실측 방법론 (라이브 키워드-실측.md v1)
- **기회점수 = 수요 ÷ log10(경쟁)**. 수요=구글/유튜브 자동완성 순위, 경쟁=유튜브 검색결과 estimatedResults(±20% 노이즈).
- 살아있는/죽은 키워드 구분. 화이트리스트 토큰 `[나라][도시][지역1~3][역명][N성급]`, 도시·지역·역명 중 하나 필수 포함.
- 도시 대표어(오사카 호텔 226만) = 못 이겨도 비용 0이라 넣음. 성급 어형 = 자동완성 없어도 AI검색용 1개 유지.
- 띄어쓰기/붙여쓰기 페어(1.5배 벌어지면 둘 다, 붙여쓰기 경쟁 500 미만이면 죽은 어형 제외). 호텔명 규칙.

## 데이터 (핵심 전제)
- **실측 = 월별 스냅샷 누적**(지금은 단발 CSV → 매달 실측 돌려 쌓아야 트렌드·선행·여러 해 비교 가능).
- 도시별 CSV 존재: 교토·나고야·도쿄·오사카·후쿠오카(`_content/youtube/keywords/`).
- **회수 = 키워드 메뉴 시계열**(성과표 아님 — 성과표엔 키워드별 조회 없음, 조회수는 영상 단위).

## 리드타임 (확정)
콘텐츠를 **3달 미리** 만들어 예약 건다. 착수 = 겨냥(피크) − 3달. 선행 뷰 = 4개월 앞까지.

## 출구 · 연동

- **연동**: 채널 규격 §4(태그란 템플릿) · 올리기(원고 태그 render §4) · 전략(추천→기획) · 호텔(도시·호텔명 키워드). 실측 소스=구글/유튜브 자동완성(수요)·검색 estimatedResults(경쟁).

## 전략 기획 큐 미리보기 (전략 메뉴 소유 · 여기선 출구로 미리봄)
