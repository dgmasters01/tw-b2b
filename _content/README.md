# `_content/` — 콘텐츠 규칙 서랍

> 여기는 **콘텐츠를 만드는 법**을 적어두는 곳입니다.
> 사업 방향(`BUSINESS.md`)도 아니고, 개발 방법(`_os/`)도 아닙니다.

---

## 왜 따로 있나

| 서랍 | 무엇이 들어 있나 | 읽는 사람 |
|---|---|---|
| `BUSINESS.md` | 얼마 받나 · 환불 조건 · 누구한테 파나 | 대표님 |
| `_os/` | Claude가 일하는 법 · 개발 규칙 | Claude |
| **`_content/`** | **제목 · 태그 · 키워드 · 파일명 규칙** | **대표님 · 직원** |

콘텐츠 규칙은 코드가 아닙니다. 그리고 채널이 늘어나면 문서도 늘어납니다.
`_os/` 안에 섞으면 개발 문서와 뒤엉키므로 따로 둡니다.

---

## 안에 무엇이 있나

```
_content/
  └─ youtube/
       ├─ 여행능력자들.md   (한국어 · 숏폼)
       ├─ 호텔이야.md       (한국어 · 롱폼)
       └─ 호텔이곳.md       (한국어 · 롱폼)
```

앞으로 채널이 늘어나면 파일이 늘어납니다. 코드는 안 바뀝니다.

---

## 누가 읽나

- **직원**: 원고를 올릴 때 이 규칙대로 결과가 나옵니다
- **Claude**: `api/youtube.js` 가 이 규칙을 읽어서 제목·태그를 만듭니다
- **대표님**: 규칙을 바꾸고 싶으면 이 파일만 고치면 됩니다
