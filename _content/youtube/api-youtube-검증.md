# api/youtube.js — 검증 기록

> 만든 날: 2026-07-09
> 무엇을 확인했나: 원고 3개를 넣어서, 손으로 만들어 둔 정답과 같은 결과가 나오는지.

---

## 무엇을 만들었나

원고 파일 하나를 넣으면 유튜브에 올릴 한 벌이 통째로 나옵니다.

```
원고(docx) ─▶ 채널 판별(cid) ─▶ 제목 · 본문 · 해시태그 · 키워드 · 파일명 · 나레이션
```

규칙은 코드에 없습니다. `_content/youtube/` 문서 3개를 읽어서 씁니다.
**문서를 고치면 결과가 바뀝니다. 코드는 안 고쳐도 됩니다.**

| 새로 만든 파일 | 하는 일 |
|---|---|
| `api/youtube.js` | 창구 (POST 하면 결과가 나옴) |
| `api/_lib/docx-text.js` | docx 를 글자로 바꿈 |
| `api/_lib/youtube-rules.js` | 규칙 문서 3개를 읽음 |
| `api/_lib/youtube-parse.js` | 원고에서 도시·지역·성급·호텔·가격을 뽑음 |
| `api/_lib/youtube-render.js` | 채널 규격대로 결과를 만듦 |

기존 파일은 하나도 건드리지 않았습니다. `package.json` 도 그대로입니다.
(라이브러리를 새로 넣지 않으려고, docx 여는 기능을 직접 만들었습니다.)

## 고친 것 — 지역 해시태그에 `호텔`을 붙였습니다

처음 돌렸을 때 12개 중 1개가 안 맞았습니다. 코드 문제가 아니라 `여행능력자들.md` 안에서
해시태그 틀과 예시가 서로 달랐습니다.

| | 고치기 전 | 고친 뒤 |
|---|---|---|
| 틀 8번째 칸 | `#[지역3]` | **`#[지역3]호텔`** |
| §9 도쿄 예시 | `#니시신주쿠` | **`#니시신주쿠호텔`** |
| §8 오사카 예시 | `#신사이바시호텔` | 그대로 (원래 맞았음) |

**대표님 결정 (2026-07-10):**
> 지역명만 쓰면 여행·맛집 시청자까지 섞인다. `지역+호텔`이 예약 의도를 가진 시청자만 걸러낸다.

**문서만 고쳤고 코드는 한 줄도 안 고쳤습니다.** 다시 돌리니 12개 전부 맞았습니다.
규칙을 문서에서 읽게 만든 이유가 바로 이겁니다.

## 시스템이 원고에서 잡아낸 잘못 (실제로 있던 것)

오사카 난바 원고에서 두 가지가 걸렸습니다.

1. **가격이 오름차순이 아닙니다.** TOP1 145,000원 / TOP2 185,000원 / TOP3 165,000원
   → 규칙(`호텔이야.md` §6-2)은 탑원 < 탑투 < 탑쓰리 입니다.
2. **챕터와 본문의 호텔 이름이 다릅니다.**
   챕터 `호텔 몬토레 그라스미아 오사카` / 본문 `호텔 몬테레 그라스미어 오사카`

도쿄 원고에서는 파일명의 `10만원 미만` 을 규칙대로 `10만원이하` 로 바꿨습니다.

## 원고에 없어서 채울 수 없는 것

원고에는 **아고다 링크도, cid 도, 평점도 없습니다.** 지어내지 않습니다.

| 없는 것 | 어떻게 하나 |
|---|---|
| 아고다 링크 (cid·hid) | 부를 때 `links` 로 넣어줍니다. 안 넣으면 `[원고 확인 필요]` |
| cid | 안 주면 원고 본문의 `호텔이곳에서` / `여행능력자들.shop` 으로 알아냅니다 |
| 평점 (호텔이야만 필요) | `[원고 확인 필요]` 로 나오고 경고합니다 |
| 지역 2·3번째 (해시태그용) | 파일명에 있으면 씁니다. 없으면 `extras` 로 넣어줍니다 |
| 키워드 실측 어형 (`우메다 대욕장 호텔` 등) | 자동완성 조사가 필요합니다. `extras.keywords` 로 넣어줍니다 |

그래서 **키워드가 300~330자로 나옵니다.** 권장은 400~450자입니다.
부족한 100자는 자동완성 실측이 필요한 부분이라 코드가 만들 수 없습니다. (다음 단계 몫)

## 어떻게 쓰나

```
POST https://gohotelwinners.com/api/youtube
헤더  x-ops-token: <CLAUDE_OPS_TOKEN>
본문  {
  "filename": "001 9월 3일~9월 5일, 2명, 일본 나고야 나고야역 근처 10만원 이하 3성급 호텔.docx",
  "docxBase64": "UEsDBBQ...",
  "cid": "1946819",
  "links": { "top1": "...cid=1946819&hid=111", "top2": "...", "top3": "..." }
}
```

나오는 것: `title` · `titles[]`(1안/2안) · `filename` · `description` · `hashtags[]` ·
`keywords` · `narration` · `publicationRow`(장부에 그대로 넣는 한 줄) · `warnings[]`

`GET /api/youtube?channels=1` 로 채널·cid 목록만 볼 수도 있습니다.

`warnings` 는 **사람이 봐야 하는 곳** 목록입니다. 비어 있으면 그대로 올리면 됩니다.
