## 1. 파일명

### 형식 (확장자 제외)
```
[도시]호텔_[도시]호텔추천_[지역]호텔_[지역]호텔추천_[도시]가성비호텔_여행능력자들
```

### 예시
```
오사카호텔_오사카호텔추천_난바호텔_난바호텔추천_오사카가성비호텔_여행능력자들
도쿄호텔_도쿄호텔추천_신주쿠호텔_신주쿠호텔추천_도쿄가성비호텔_여행능력자들
```

### 규칙
| 항목 | 값 |
|---|---|
| 구분자 | `_` 언더바 |
| 상한 | 한글 60자 / 180바이트 |
| 주키워드 | 5개 (최대 6개) |
| 금지 문자 | `-` `,` 공백 `\ / : * ? " < > \|` |
| 적용 | 영상(.mp4) · 썸네일(.jpg) 동일 |
| 지역 2개일 때 | 첫 번째 지역만 사용 |

### 근거
- 유튜브는 원본 파일명을 **웹에 노출하지 않는다** (HTML 전수 확인). URL은 무작위 ID.
- 따라서 SEO 효과 **없음**. 하이픈/언더바 차이도 무의미.
- 목적은 **자산 관리 + 자동화 파싱**. 언더바가 스크립트 친화적.
- ⚠️ 파일명이 업로드 시 기본 제목이 되므로, **반드시 확정 제목으로 덮어쓸 것**.

## 2. 제목

### 1안 — 성급 미표기
```
[나라] [지역] [가격대] [도시] 호텔 추천! [도시] 가성비 호텔 TOP3 #[지역] #[도시]호텔
```

### 2안 — 성급 반영 (4성급 이상에만 발동)

**4성급 이상 → 제목 앞쪽 삽입**
```
[나라] [지역] [가격대] [N성급] [도시] 호텔 추천! [도시] 가성비 호텔 TOP3 #[지역] #[도시]호텔
```

**3성급 이하 → 제목에서 제외.** 1안만 사용. 키워드란·설명란 해시태그에만 넣는다.

> [v2 변경] 실측 결과 3성급은 수요가 적은데 경쟁이 더 높다.
> 도쿄 3성급 호텔 55,352 vs 도쿄 4성급 호텔 36,521 / 오사카 3성급 15,505 vs 4성급 14,696
> 또한 `도쿄 3성급`을 치면 자동완성이 4성급을 권한다 = 3성급 검색자가 4성급 영상을 본다.
> 5성급은 앞쪽 삽입 (경쟁 최저, 프리미엄 소구).

### 규칙
- 가격대는 **붙여쓰기**: `10만원이하` / `10만원대` / `20만원대`
- **`미만` 금지** → `이하` 사용. 자동완성에 `10만원 미만 호텔`이 존재하지 않는다.
  (도쿄·오사카 모두 `10만원대` 1위 / `10만원 이하` 2위 / `미만` 없음)
- 지역 2개면 첫 번째만 제목에, 나머지는 설명란·태그로
- 해시태그는 제목 끝 고정
- 100자 이내 (유튜브 제목 한도)
- 업로드 시 1안·2안 함께 제시 → 대표님 선택

### 확정 예시
```
[도쿄 · 3성급] 일본 신주쿠 10만원이하 도쿄 호텔 추천! 도쿄 가성비 호텔 TOP3 #신주쿠 #도쿄호텔
[오사카 1안] 일본 난바 10만원대 오사카 호텔 추천! 오사카 가성비 호텔 TOP3 #난바 #오사카호텔
[오사카 2안] 일본 난바 10만원대 4성급 오사카 호텔 추천! 오사카 가성비 호텔 TOP3 #난바 #오사카호텔  ← 4성급이므로 2안 발동
```

## 3. 설명란

### 블록 순서
```
1. 후킹 한 줄
2. 선정 기준 + (가격은 M/D~M/D N인 기준)
3. ━━━ TOP1 → TOP2 → TOP3 + 예약 링크 ━━━
4. ※ 가격 변동 고지
5. ※ 어필리에이트 고지
6. 구독 유도 + 대표 링크
7. AI 검색용 위치 키워드 문단
8. 해시태그 12개
```

### 블록 1 — 후킹
```
[도시] [지역] [가격대] 가성비 [N]성급 호텔 TOP3 추천합니다.
```

### 블록 2 — 선정 기준 (문서마다 달라짐)
```
[조건1], [조건2], [조건3]을 기준으로 [지역] [N]성급 호텔을 비교해 선정했어요. (가격은 M/D~M/D N인 기준)
```
- 조건 3개 = 원본 문서에서 **3개 호텔이 공통 언급한 항목** 상위 3개 추출
- 조건은 **명사형**으로 (`조용함` ✗ → `조용한 주변 환경` ✓)
- 지역·성급을 본문에서 한 번 더 언급 → AI 문맥 확정
- 조건 후보 풀: 입지 / 치안·환경 / 편의 / 객실 / 부대시설

### 블록 3 — 호텔 블록
- 순서: **TOP1 → TOP2 → TOP3** (전 채널 통일, 전환 최적화)
- **개별 가격 표기 안 함** (조회 시점 변동)
- 각 블록 끝: `👉 예약: [아고다 링크]`
- 서술 관점: **여행 동선 · 위치 · 안전**

### 블록 4·5 — 고지 (고정)
```
※ 호텔 가격은 보시는 시점에 따라 상이할 수 있습니다.
※ 해당 링크로 예약하시면 아고다에서 여행능력자들에게 소정의 수수료를 지급하며, 더 좋은 콘텐츠를 만드는 원동력이 됩니다.
```

### 블록 6 — 구독 유도
```
🏨 매일 새로운 [도시]·[타도시]·[타도시] 등 가성비 호텔 추천을 받아보고 싶다면 구독!
🔗 전체 호텔 예약 링크: www.여행능력자들.shop
```

### 블록 7 — AI 검색용 위치 키워드 (5곳)
```
이 영상은 [지역1], [지역2], [지역3], [역명], [도시] 주변 여행을 준비하는 분들을 위한
가성비 숙소 가이드입니다. [도시] [N]성급 호텔, [역명] 근처 숙소,
[지역2] 도보권 가성비 호텔을 한 번에 비교하세요.
```

### 블록 8 — 해시태그 12개
```
#[지역1]호텔 #[도시]호텔추천 #[도시]여행 #[도시]가성비호텔 #[도시][N]성급 #[역명]호텔 #[지역2]호텔 #[지역3]호텔 #[도시]숙소 #[도시]호텔위치 #[나라]여행 #여행능력자들
```
- **노출되는 앞 3개**: 지역호텔 → 도시호텔추천 → 도시여행
- 15개 초과 시 전체 무효 (유튜브 정책)
- 브랜드명 · `#[도시]숏폼` 제외
- `#호텔추천` 금지 → `#[도시]호텔추천`으로 도시 한정
- **지역 태그는 반드시 `호텔`을 붙인다** (`#니시신주쿠` ✗ → `#니시신주쿠호텔` ✓)
  지역명만 쓰면 여행·맛집 시청자까지 섞인다. `지역+호텔`이 예약 의도를 가진 시청자만 걸러낸다. (2026-07-10 대표님 확정)

## 5. 키워드 (유튜브 스튜디오 태그란, 500자)

### 5계층 구조
| 계층 | 개수 | 역할 |
|---|---|---|
| L1 대키워드 (나라·도시 여행) | 5 | 광역 |
| L2 도시 + 호텔/숙소 | 15 | 주력 |
| L3 지역 키워드 | 12 | **전환 핵심** |
| L4 조건·성급 | 3 | AI 검색 |
| L5 호텔명 | 3~6 | **최고 전환** (경쟁 159~657건) |

### 슬롯 템플릿
```
[나라]여행, [도시]여행, [도시] 여행, [도시]자유여행, [도시] 자유여행,
[도시]호텔, [도시] 호텔, [도시]호텔추천, [도시] 호텔 추천,
[도시]가성비호텔, [도시] 가성비 호텔, [도시] 호텔 가성비, [도시] 호텔 위치, [도시]호텔예약,
[도시]숙소, [도시] 숙소, [도시]숙소추천, [도시] 숙소 추천, [도시] 숙소 가성비, [도시] 숙소 위치,
[지역1]호텔, [지역1] 호텔, [지역1]호텔추천, [지역1] 호텔 추천, [역명] 호텔, [지역1]숙소, [지역1] 숙소,
[지역2]호텔, [지역2] 호텔, [지역3] 호텔,
[도시] [N]성급 호텔,
[호텔명1], [호텔명2], [호텔명3]
```

### 규칙
1. **400~450자** 채움 (500자 꽉 채우면 희석)
2. **띄어쓰기·붙여쓰기 페어로** 투입 — 유튜브 자동완성이 별개로 취급
3. **지역 키워드가 전체의 40%** — 전환의 핵심
4. 유튜브 **자동완성에 없는 어형은 배제** (예: `요츠야 호텔` = 죽은 키워드)
5. 호텔명은 **자동완성 검증 후 채택**. 다른 호텔과 이름 겹치면 금지
   - 예: `호텔 몬테레이 오사카`는 그라스미어와 다른 호텔 → 제외

### 근거
- 유튜브 공식: 태그는 발견에 **최소 역할**, 주로 **철자·표기 변형 보완** 용도
- 한국어 띄어쓰기 변형이 정확히 그 케이스
- 다만 태그란은 `<meta name="keywords">`로 HTML에 노출됨 → **구글·AI 검색이 읽음**

## 6. AI 검색 최적화

### 실제 읽히는 요소 (우선순위)
| 요소 | 웹 노출 | 활용도 |
|---|---|---|
| 자막·스크립트 | ✓ | ★★★ 최고 |
| 설명란 첫 200자 | ✓ | ★★★ |
| 제목 | ✓ | ★★★ |
| 키워드(meta) | ✓ | ★★ |
| 파일명 | ✗ | 0 |

### 원칙
1. 첫 두 줄 안에 핵심 답 완결
2. 가격·거리는 숫자로 명시 (형용사는 인용 불가)
3. 조건 키워드는 검색 질의 어형과 일치
4. 하단 위치 키워드 5곳 문단으로 인근 지역 커버

## 7. 키워드 리서치 방법

### 도구
- 유튜브 자동완성 API: `suggestqueries.google.com/complete/search?client=firefox&ds=yt&hl=ko&gl=kr&q=`
- 경쟁도: 유튜브 검색결과 HTML의 `"estimatedResults"`

### 지표
| 지표 | 의미 | 주의 |
|---|---|---|
| 자동완성 순위 | 수요 프록시 | **띄어쓰기 구분함** |
| 파생어 개수 | 관심 폭 | |
| 경쟁 영상 수 | 경쟁 프록시 | **띄어쓰기 정규화됨**, ±20% 노이즈 |
| 기회점수 | 수요 ÷ log10(경쟁) | 자체 산식 |

### 검증된 사실
- `오사카 호텔` = `오사카호텔` = 경쟁 226만 (동일. 짧은 조합은 유튜브가 자동 교정)
- `오사카 가성비 호텔` 151,394 vs `오사카가성비호텔` **14,554** (3어절 이상은 실제로 갈림)
- 호텔명 경쟁: 히요리 159 / 몬테레 407 / 페어필드 556 → **사실상 독점 가능**

## 10. 업로드 체크리스트

5. 아고다 링크 3개 클릭 테스트

## 11. 미결 사항

| # | 항목 | 상태 |
|---|---|---|
| 1 | 영상 자막 개별 가격 표기 유지/삭제 | ⬜ 미정 |
| 2 | 나레이션 인트로 대키워드 삽입 | ⬜ 미정 |
| 3 | 마지막 자막 CTA (`👇 아래 설명란에 예약 링크 있어요`) | ⬜ 미정 |

### 미결 1·2·3 배경
- 유튜브는 **음성을 인식**한다. 나레이션에 `가성비` `N성급` `가격대` `숙소`가 없다.
- 인트로 한 문장 수정으로 대키워드 전체가 자막·음성에 들어감.
  - 현재: `여행하기 좋은 호텔 TOP3를 추천해 드릴게요!`
  - 제안: `난바에서 10만원대로 잡는 가성비 4성급 호텔 TOP3를 추천해 드릴게요!`
- 쇼츠는 설명란 링크가 **클릭 가능**하나 `...더보기` 탭이 필요 → 행동 지시어로 탭률 상승
