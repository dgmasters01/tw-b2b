### 성급 처리
| 성급 | 제목 | 조건 |
|---|---|---|
| 5성급 | O | 앞쪽 삽입 |
| 4성급 | O | 앞쪽 삽입 |
| **3성급 이하** | X | **제외** (키워드란·해시태그에만) |

**추가 조건: 해당 도시에 성급 검색 수요가 있을 때만 표기**
- 오사카 4성급 → 자동완성 실존, 경쟁 14,696 → **표기**
- 나고야 3성급 → 자동완성 0개 → **제외**
- 도쿄 3성급 → 경쟁 55,352 (4성급 36,521보다 높음) → **제외**

## 3. 본문 (더보기)

### 블록 순서
```
1. 요약 2줄
2. 🔻 최저가 링크는 아래를 참고해주세요.
3. 🏅 TOP1 → TOP2 → TOP3 + 링크
4. ═════ / ※ 기간
5. 🎬 챕터  (영상 흐름 순서: TOP3 → TOP1)
6. 🚶 주변 명소 (🔸)
7. 공항 이동 요약 문단
8. 채널 소개 + 구독 유도
9. ※ 고지 2줄
10. #해시태그 10개
```

> **링크는 TOP1이 맨 위 / 챕터는 TOP3가 맨 위.** 링크는 전환용, 챕터는 영상 흐름용. 전 채널 통일.

### 블록 1 — 요약 2줄
```
[도시] 호텔 추천을 찾는다면, [지역] 호텔 중에서 [가격대] [N성급]만 골라 정리했습니다.
[공항]에서 [소요시간] 이내, [역명] 도보권, 무료 짐 보관까지 확인한 [도시] 가성비 숙소 추천 TOP3입니다.
```

### 블록 2 — 링크 헤더 (고정)
```
🔻 최저가 링크는 아래를 참고해주세요.
```

### 블록 3 — 호텔 블록 (3줄 구성)
```
🏅 TOP1 · [한글명] ([영문명])
   [역명] 도보 O분 · [공항] OO분
   [특징1] · [특징2] · [특징3]
   ➜ 예약 바로가기: [아고다 제휴 링크]
```
- 순위 기호 `🏅 TOPN ·`
- 링크 문구 `➜ 예약 바로가기:`
- **가격 표기 안 함** / **평점 표기 안 함**
- 2행 = 위치·이동 / 3행 = 특징

### 블록 4 — 기간
```
═════════════════
※ 기간 : YYYY년 M월 D일~M월 D일 (N박·N인) 기준
```

### 블록 5 — 챕터
```
🎬 챕터
00:00 오늘의 조건 ([지역]·[가격대]·[N성급])
XX:XX TOP3 [호텔명]
XX:XX TOP2 [호텔명]
XX:XX TOP1 [호텔명]
XX:XX 최저가 확인 방법
```
- 마지막 챕터는 `최저가 확인 방법`
- 타임스탬프는 원고 그대로

### 블록 6 — 주변 명소
```
🚶 주변 명소

🔸 [호텔명]
   [스팟1] 도보 O분 · [스팟2] · [스팟3]
   [스팟4] · [스팟5]
```

### 블록 7 — 공항 이동 요약 (고정 틀)
```
추천호텔들은 [공항]에서 [교통수단]을 타고 [역명]에 내리면
XX~XX분 만에 도착합니다. [도시] 위치를 고민하시는 분들에게
이 호텔들을 추천드립니다.
```

### 블록 8 — 채널 소개 + 구독 (고정 틀)
```
호텔이곳은 [도시] 호텔 추천부터 [역명] 숙소 위치까지, 지도로 확인한 정보만 제공합니다.
구독하고 호텔 추천을 받아 보세요.
```

### 블록 9 — 고지 2줄 (고정)
```
※ 요금은 예약 날짜에 따라 변동될 수 있습니다.
※ 링크를 통해서 예약이 완료되면 아고다에서 호텔이곳 채널에 일정 수수료를 지급합니다.
```

### 블록 10 — 해시태그 10개
```
#[도시]호텔 #[도시]숙소추천 #[도시]호텔추천 #[역명]호텔 #[도시]가성비호텔 #[도시]호텔위치 #[도시][N]성급 #[도시]여행 #[나라]여행 #호텔이곳
```

- 15개 초과 시 전체 무효

- 노출되는 앞 3개: `#[도시]호텔` `#[도시]숙소추천` `#[도시]호텔추천`
