## 1. 제목

### 1안 — 지역 우선형 (추천)
```
[나라] [도시] [지역] 호텔 추천 TOP3, [N성급] [가격대] [도시] 가성비 숙소
```

### 2안 — 도시 우선형
```
[나라] [도시] 호텔 추천 [N성급], [가격대] [지역] 가성비 호텔 TOP3
```

### 슬롯 순서
```
[성급] → [가격대] → [지역/도시]
```
- 가격대는 **항상 지역·도시 바로 앞**

### 성급 처리
| 성급 | 제목 | 위치 |
|---|---|---|
| 5성급 | O | 앞쪽 삽입 |
| 4성급 | O | 앞쪽 삽입 |
| **3성급 이하** | X | **제외** (키워드란·해시태그에만) |

> 실측 근거: 도쿄 3성급 호텔 경쟁 55,352 vs 4성급 36,521 / 오사카 3성급 15,505 vs 4성급 14,696.
> 수요는 적은데 경쟁이 더 높다. `도쿄 3성급` 자동완성이 4성급을 권한다.

### 세부 규칙
| 항목 | 값 |
|---|---|
| 연도 표기 | **금지** (오래된 영상도 계속 추천되므로 조회 저해) |
| `\|` 파이프 | **금지** (한국 유튜브 관행 아님) |
| 구분자 | `,` 쉼표 |
| 제목 해시태그 | **없음** |
| 가격대 | `10만원이하` / `10만원대` — 붙여쓰기 |
| `미만` | **금지** → `이하` (자동완성에 `미만` 없음) |
| 글자수 | 60자 이내 |
| 앞 통덩어리 | 자동완성 실존 어형만 (`일본 오사카 호텔 추천`, `오사카 우메다 호텔 추천`) |

### 예시
```
[3성급 1안] 일본 오사카 우메다 호텔 추천 TOP3, 10만원이하 오사카 가성비 숙소
[3성급 2안] 일본 오사카 호텔 추천, 10만원이하 우메다 가성비 호텔 TOP3
[4성급 1안] 일본 오사카 우메다 호텔 추천 TOP3, 4성급 10만원대 오사카 가성비 숙소
[4성급 2안] 일본 오사카 호텔 추천 4성급, 10만원대 우메다 가성비 호텔 TOP3
```

## 3. 본문 (더보기)

### 블록 순서
```
1. 요약 2줄
2. 🏷️ 아래의 최저가 예약 링크입니다.
3. ▶ 1위 → 2위 → 3위 + 링크
4. ───── / ※ 조사 기준
5. ⏱️ 챕터  (영상 흐름 순서: TOP3 → TOP1)
6. 🏨 호텔 한눈에 보기 (▪)
7. 공항 접근성 요약 문단
8. 구독 유도
9. ※ 고지 2줄
10. #해시태그 10개
```

> **링크는 TOP1이 맨 위 / 챕터는 TOP3가 맨 위.** 링크는 전환용, 챕터는 영상 흐름용. 전 채널 통일.

### 블록 1 — 요약 2줄
```
[도시] 호텔 중에서도 [지역] 가성비 호텔을 찾는 분들을 위해, [가격대] [N성급] 숙소만 정리했습니다.
[역명] 도보권, [조건2], [조건3]을 기준으로 검증한 [도시] 숙소 TOP3입니다.
```
- `추렸습니다` 금지 → **`정리했습니다`**

### 블록 3 — 호텔 블록
```
▶ 1위 · [한글명] ([영문명])
   [역명] 도보 O분 · 1박 약 OO,OOO원 · 평점 O.O · [공항] 약 OO분
   🔗 최저가 확인: [아고다 제휴 링크]
```
- 순위 기호 `▶ N위 ·` (여행능력자들 `🥇`와 구분)
- 링크 문구 `🔗 최저가 확인:` (여행능력자들 `👉 예약:`과 구분)
- **가격·평점 표기함** (여행능력자들은 표기 안 함)

### 블록 4 — 조사 기준
```
─────────────────
※ 조사 기준: M월 D일~M월 D일(N박·N인) 아고다 기준
```

### 블록 5 — 챕터
```
⏱️ 챕터
00:00 오늘의 조건 ([도시] [지역]·[가격대]·[N성급])
XX:XX TOP3 [호텔명]
XX:XX TOP2 [호텔명]
XX:XX TOP1 [호텔명]
XX:XX 예약 방법
```
- 마지막 챕터는 `예약 방법` (`정보 요약 &` 제거)
- 숏폼에는 챕터 기능이 없으므로 여행능력자들에는 넣지 않는다

### 블록 6 — 호텔 한눈에 보기
```
🏨 호텔 한눈에 보기

▪ [호텔명]
  [시설·스펙·주변 명소 2~3줄]
```

### 블록 9 — 고지 2줄 (고정)
```
※ 가격은 조사 시점 기준이며, 예약일과 객실 조건에 따라 달라질 수 있습니다.
※ 위 링크를 통해 예약이 이루어지면 아고다로부터 일정 수수료가 호텔이야 채널에 지급되며, 더 나은 호텔 정보를 제작하는 데 사용됩니다.
```
> 여행능력자들 고지와 문장 구조·어휘가 전부 다르다. `시청자 부담 없음` 류 문구는 넣지 않는다.

### 블록 10 — 해시태그 10개
```
#[지역]호텔 #[도시]호텔추천 #[도시]가성비호텔 #[지역][특징]호텔 #[역명]호텔 #[도시][N]성급 #[도시]숙소 #[도시]호텔위치 #[나라]여행 #호텔이야
```
- 15개 초과 시 전체 무효
- 여행능력자들과 겹쳐도 무방. **채널명 태그만 교체**

## 10. 업로드 체크리스트

1. 영상·썸네일 파일명 변경
2. 업로드 → **기본 제목을 확정 제목으로 덮어쓰기**
3. 설명란 붙여넣기 (해시태그 10개 포함)
4. 스튜디오 → 더보기 → 태그란에 키워드 붙여넣기
5. 챕터 타임스탬프가 실제 영상과 맞는지 확인
6. 아고다 링크 3개 클릭 테스트 (cid=1932026 확인)
