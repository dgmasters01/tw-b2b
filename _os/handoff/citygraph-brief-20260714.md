# 도시그래프 작업 브리프 (인계용)

> 채팅 끊김 대비 저장본. 확정문서 = D-065. 저장 2026-07-14.

## 이게 뭔가
키워드 메뉴 **도시 상세 ⑤ "그 도시 수요 트렌드"** 그래프.
월별 키워드 실측(수요)을 쌓아 계절 패턴을 보여주는 선그래프.
→ "어느 도시가 언제 뜨는지" 미리 안다.

## 그래프 모양
- 가로 = 1~12월 / 세로 = 수요(기회점수 지수)
- 여러 해(2024·2025·2026) 겹쳐 그림 → 계절 피크 눈에 보임
- 예: 오사카=봄·가을, 방콕=겨울, 다낭=여름

## 데이터
- 월별 실측 스냅샷 누적 (기회점수 = 수요 ÷ log10(경쟁))
- 지금은 단발 CSV(`_content/youtube/keywords/{도시}.csv`) → 매달 쌓아야 트렌드 완성
- 회수 = 키워드 시계열(성과표 아님)

## 어디에 있나
키워드 → 도시 상세: ①추천 ②순위 ③태그란 ④미개척 **⑤수요트렌드 ← 이번 작업**

## 지금 라이브 상태 (studio.html 실측)
- 키워드 메뉴 = 기본 리스트만 라이브(`/api/content-keywords`, view-keyword)
- 도시 상세 · 수요 트렌드 그래프 = **아직 없음** → 이번 대상
- 중복 점검: 키워드 쪽 트렌드/차트 코드 없음(성과표 st-pf-trend는 별개 메뉴)
