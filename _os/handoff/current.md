## 📐 확정된 설계 (D-053)

- **admin 호텔 상세 = 매니저 분석 형태를 미러링, 별도 페이지로 분리** (허브 목록 → 호텔 클릭 → 상세). 탭에 욱여넣지 않음.
- **탭 = 개요 · 채널별 · 패턴 · 예약 상세** (비교용 나라/도시/호텔/성급 탭은 단일 호텔 상세에 안 맞아 제외).
- **예약 컬럼 = 매니저 페이지 그대로**: 예약번호 · 예약일 · 체크인 · 체크아웃 · 금액 · 수수료 · 상태 · 기기.
- **수수료 2버전**: 매니저 화면엔 빼고(우리 제휴수익 비노출), admin 화면엔 포함. 같은 컴포넌트 + 표시 플래그.
- **기간 구분 4단계**: 마케팅 전 / 마케팅 기간 / 마케팅 후 / 전체. "전체"는 분석 메뉴 "전체"와 같은 소스로 **동기화**.
- **회차** = `campaign_log` 기반 회차 선택. 마케팅 기간/후 구분은 각 회차 6개월 기간으로 예약 자동 분류.
- **기존(마케팅 전) 예약** = 과거 데이터에서 **호텔명+도시+국가 매칭** → 자동매칭 후 **대표님이 연결 확정**(오매칭 방지). 강력한 영업 자료.
- **담당 매니저 이름·연락처** = 가입 폼 필드 추가 + 상세 페이지 머리에 표시.
