## 메모리 16 — i18n 작업 운영 룰

[TW B2B i18n 작업 운영 룰] 한국어 번역(data-ko 속성, JS T.lang 분기)은 영어 본체 페이지가 모두 완성된 후 **단일 i18n 일괄 작업**으로 처리한다. 신규 페이지/탭/모달 개발 시에는 영어만 작성하고 data-ko / 한국어 분기 코드 추가하지 말 것 — 매번 추가하면 누락 리스크 큼. 한국어 토글 버튼은 작동 보장하지만, 번역 데이터는 일괄 작업 시점까지 부분 노출 상태로 둔다. 단일 i18n 작업은 별도 작업 단위로 명시 ("Phase 3 i18n 일괄 적용" 같은 작업명)로 받아서 처리.
