## 규칙 (부칙 16.1-ⓓ "거대 파일" 확장 — 읽기에도 적용)
1. **통째 다운로드/read/cat/내용 grep 금지.** 큰 파일은 먼저 `wc -l`·`awk 'length>100000{print NR}'`로 괴물 줄을 찾는다.
2. 검색은 **줄번호만**: `grep -niE '패턴' file | cut -d: -f1`. 내용 출력(`grep` 기본)은 괴물 줄에 걸리면 폭발.
3. 읽기는 **구간만**: `sed -n 'A,Bp' file` 또는 view의 line range. 괴물 줄 번호는 범위에서 제외.
4. 수정은 해당 줄/블록만 부분 편집. 거대 데이터 줄은 건드리지 말고 SHA 보존 검증.
5. 거대 파일 1개 작업은 한 채팅에 그것만 (부칙 16.1-ⓓ).
