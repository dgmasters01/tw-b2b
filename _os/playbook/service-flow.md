## 🎯 전체 흐름 한눈에

```
[Stage 0] 공개 진입 → 집합 자랑 노출
   ↓
[Stage 1] 회원가입 → 채널 자랑 + 이메일 인증
   ↓
[Stage 2] 호텔 정보 입력 → Agoda 매칭 (3단계 안전망)
   ↓
[Stage 3] 관리자 승인 → 본인 호텔 데이터 확인 가능
   ↓
[Stage 4] 영업 페이지 (결제 유도) → 4중 안전 메시지 + Extranet 검증
   ↓
[Stage 5] PayPal $200 결제 → status=paid
   ↓
[Stage 6] 영상 제작 (3~7일) → 8개 채널 노출
   ↓
[Stage 7] 월간 리포트 (매월) → PDF + 메일
   ↓
[Stage 8] D-30 알림 (5개월 차) → 재계약 안내
   ↓
[Stage 9] D-Day (6개월 차) → 0건 환불 또는 재계약
```

## 📋 Stage별 상세

### Stage 0: 공개 진입

**페이지**: `index.html` (공개)
**매니저가 보는 것**:
- 집합 자랑 헤드라인: "**총 3,774건 예약 / $854K 누적 매출 / 200+ 호텔이 우리 채널로 영상 받음**"
- "당신 호텔도 우리 채널에 영상 노출시키세요" CTA
- "결제 버튼" 클릭 시 → "결제는 가입 후 가능" 모달 → `signup.html` 이동

**약관 안전성**: ✅ 집합 자랑 (식별 불가능) — 조항 4.1.1 권장 행동
**시스템 박힘**: 매출 통계는 `BL-PAST-VIDEO-RECON`이 생성하는 누적 데이터 활용

---

### Stage 1: 회원가입

**페이지**: `signup.html`
**입력 정보 (D-032 확정)**:
- 이메일 (인증 필수)
- 비밀번호
- 호텔 이름
- **국가 (필수)** — 동남아 1차 타겟 분기용
- 매니저 본인 이름 + 연락처

**가입 모달에 박는 메시지** (D-038 4중 안전 구조 ②):
- "9M+ 조회수 · 6개 언어 · 8개 YouTube 채널"
- "5개 OTA 파트너 (Agoda, Trip.com, KKday, Expedia, Booking.com)"
- "6개월 보장 + 0건 시 100% 환불"

**자동 메일** (SQ-G, BL-EMAIL-LOCALE-ROUTING):
- 가입 확인 메일 (영어 default, 국가별 분기 예정)
- 인증 링크 클릭 → status: pending → verified

**약관 안전성**: ✅ 채널 자랑 (우리 자체 데이터)

---

### Stage 2: 호텔 정보 입력

**페이지**: `hotel-info.html`
**입력 정보**:
- 호텔 영문명 (Google Maps 우선)
- 영문 주소
- 호텔 등급 (자동 매칭 보조)
- 객실 수

**Agoda 매칭 3단계 안전망** (2026-04-26 결정, 변경 불가):
1. **1단계 자동 매칭**: 호텔명 + 주소 → Agoda 데이터베이스 검색
2. **2단계 보조 매칭**: 매니저가 제공한 booking URL/Trip URL 검증
3. **3단계 수동 매칭**: 관리자가 직접 확인 (status: review)

**핵심 원칙 (BUSINESS.md 15-A)**: 매칭 실패해도 **100% 활성화**. 매출 흐름 막지 않음.

**자동 메일**:
- 호텔 정보 접수 알림 (관리자 + 매니저)

---

### Stage 3: 관리자 승인

**관리자 페이지**: `admin.html` (또는 `admin-gohotel-XXX.html`)
**관리자 동작**:
- 호텔 정보 확인 + Agoda 매칭 결과 검토
- 승인 → status: approved → 매니저에게 메일 (SQ-G)

**매니저에게 보내는 메일 (SQ-G `hotel-status-notify` endpoint)**:
- 제목: "Your hotel is approved — next step is payment"
- 본문:
  ```
  Hi [매니저 이름],

  We've reviewed your hotel listing and you're approved!

  Next step: Visit your dashboard to see what your hotel could expect
  and make payment when ready.

  → [Dashboard Link]
  ```

---

### Stage 4: 영업 페이지 (결제 유도) ⭐ 핵심

**페이지**: `sales.html` 또는 `dashboard.html` (결제 전)
**매니저가 보는 것** (D-038 4중 안전 구조 ③+④):

```
=========================================
당신 호텔: OO Hotel Da Nang (다낭)
status: approved (결제 대기)
=========================================

[상단 — 집합 자랑]
"TravelWinners B2B는 누적 3,774건 예약 / $854K 매출"

[중단 — 본인 호텔 데이터 (D-035 3구간 분기)]

  Case A — 누적 $1,000+:
  "당신 호텔은 우리 채널을 통해
   지난 6개월 동안 12건 예약 발생 ($2,340 누적)
   ─ AGD-2026-001234 (2026-03-15)
   ─ AGD-2026-001456 (2026-04-02)
   ...

   $200 결제하시면 추가 6개월 더 노출.
   기존 결과 보면 본전 회수는 보장 수준."

  Case B — 누적 $200~$999:
  "이미 우리 채널에서 N건 예약 발생.
   6개월 보장으로 확장 시 더 많은 매출 기대."

  Case C — 누적 $0~$199:
  (D-035 숨김 정책)
  "9M 조회수 채널에 당신 호텔 영상 만들어드림.
   6개월 동안 예약 0건이면 100% 환불."

[하단 — 검증 가이드 (D-038 ④)]
"위 예약번호를 당신 Agoda Extranet에서 검색하시면
 실제 예약 발생 사실을 즉시 확인 가능합니다.

 (참고: 검증은 호텔 본인 Agoda 권한으로 자유롭게 가능)"

[결제 버튼]
🟢 PayPal $200 결제하기
```

**매니저 자주 묻는 질문 예상**:

| "왜 Agoda 통해서?" | "Agoda는 글로벌 1위 OTA. 우리는 마케팅 채널 제공, 예약은 Agoda 인프라." |
| "0건이면 진짜 환불?" | "6개월 후 0건이면 자동 100% 환불. 별도 신청 불필요." |
| "다른 호텔도 보여줘" | "다른 호텔 데이터는 보안 정책상 비공개. 당신 호텔만 확인 가능." |

**약관 안전성**: ✅ 4중 안전 (집합 + 채널 + 본인 + Extranet)

### Stage 6: 영상 제작 (3~7일)

**Day 0~3**:
- 영상 제작팀이 호텔 정보 기반 영상 제작
- 영상 설명란에 Agoda affiliate 링크 박음 (commission 발생 기반)

**Day 4~7**:
- 8개 채널 동시 노출 (한/일/영/베트남어/중국어/태국어 등)
- status: paid → producing → published

**자동 메일** (SQ-G):
- 영상 첫 노출 알림 (5개 채널 링크 첨부)

---

### Stage 7: 월간 리포트 (매월 결제일)

**페이지**: `marketing.html` (성과 대시보드)
**매니저가 보는 것**:
- 이번 달 영상 조회수
- 우리 자체 추적: 영상 설명란 클릭 수
- 본인 호텔 발생 예약 N건 (예약번호 리스트)
- 6개월 보장 D-Day 카운트
- PDF 보고서 다운로드 버튼 ⭐ (매니저 핵심 기능)

**자동 메일**:
- 매월 결제일에 PDF 첨부 메일 (월간 리포트)

### Stage 9: D-Day (6개월 차)

**Case A — 예약 0건**:
- 자동 환불 트리거 발동

- status: paid → refunded
- 자동 메일: "0건이라 100% 환불 처리됨"

**Case B — 예약 발생**:
- 재계약 권유 메일
- "동일 가격 $200으로 6개월 더 연장"
- 매니저 클릭 → 재결제 → 새 6개월 사이클

## 🚨 매니저 컴플레인 대응 시나리오

### "예약이 안 났어요!"

대응:
1. marketing.html에서 본인 호텔 예약 확인 (없으면 표시 안 됨)
2. 영상 조회수 + 클릭 수는 노출 (마케팅은 한 것)
3. D-Day까지 기다리시면 0건 시 100% 환불 자동 처리
4. "지금까지 N건 클릭 발생했고 D-Day까지 남은 기간 있으니 기다려달라"

### "Agoda commission 얼마 받았어요?"

대응:
- "그건 우리와 Agoda 간 비즈니스 영역으로 공개 불가 (NDA)"
- "당신은 우리 영상 마케팅 서비스 $200 + 6개월 보장 받음. 이게 우리 계약 영역."

### "다른 호텔은 얼마나 벌었어요?"

대응:
- "다른 호텔 데이터는 보안 정책상 비공개"
- "전체 누적은 3,774건/$854K (집합 통계는 공개)"
