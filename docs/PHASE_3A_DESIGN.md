## 6. 우선순위 자동 정렬 알고리즘

```python
def calculate_priority_score(task):
    score = 0
    
    # 1. 명시 우선순위
    score += {"P0": 100, "P1": 70, "P2": 40, "P3": 20}.get(task.priority, 0)
    
    # 2. 블로커 여부 (다른 작업 막으면 +50)
    if task.is_blocking_others():
        score += 50
    
    # 3. 마감일 임박
    days_left = (task.deadline - now).days
    if days_left <= 1: score += 30
    elif days_left <= 3: score += 20
    elif days_left <= 7: score += 10
    
    # 4. 진행 중 우선 (+25)
    if task.status == "in_progress":
        score += 25
    
    # 5. 막힘 페널티 (-15)
    if task.status == "blocked":
        score -= 15
    
    # 6. 사이즈 영향
    score += {"small": 5, "medium": 0, "large": -10}.get(task.size, 0)
    
    return score
```

**두 그룹 분리 정렬:**
- **그룹 A** (대표님 결정): owner=human → 점수 내림차순 → 상위 5건 표시
- **그룹 B** (Claude 자율): owner=autonomous → 점수 내림차순 → 상위 10건 표시
