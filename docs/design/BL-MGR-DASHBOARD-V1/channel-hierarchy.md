## 2. 위계 표시 원칙

### UI 표시 순서 (모든 화면)
- **한국 박스 = 위** / **해외 박스 = 아래**
- 한국 = 메인 진입 / 해외 = 글로벌 확장
- 한국 박스가 더 크거나, 같은 크기라도 위 정렬

### 동급 표기 금지
- ❌ "8채널에 노출됩니다" (한국·해외 동급으로 보임)
- ✅ "한국 4채널 (메인) + 해외 4채널 (글로벌)" (위계 명시)

### 매출 표기 원칙 (v2 추가)
- **8채널 통합 누적 = 3,774건 / $854K** (sales.html · 헤더 Proof)
- 매니저 개별 대시보드:
  - 호텔별 예약은 채널 구분 없이 통합 표시 (아고다 시트가 통합 집계라서)
  - 한국/해외 박스에 각각 "운영 중 N개 채널" 표시
  - "8채널 모두에 당신 호텔이 노출 중" 메시지

## 3. 화면별 적용

### 매니저 대시보드 홈 탭
헤더 신뢰 자산 아래 메인 약속 카드:
```
🥇 한국 4채널 (메인 진입) — 운영 중 · 호텔이야 한국·호텔이곳·호텔닷컴·여행능력자들
🥈 해외 4채널 (글로벌 확장) — 운영 중 · Koreahotelguide·호텔이야 일본·世界就是家·Korea Hotel 베트남어
```

매출은 호텔 단위로 통합 표시 (채널별 분리 X — 아고다 시트가 통합이라):
```
이 호텔 총 예약: 12건 / $8,420
8채널 모두 운영 중
```

### 영상 탭 채널 표
- 한국어 4채널을 위에, 다국어 4채널을 아래로 정렬
- 라벨: 한국 = "메인 진입" / 해외 = "글로벌 확장"
- 상태 컬럼: 8채널 모두 "노출 중" 뱃지 가능

### 고객 분석 탭
채널별 분석 박스 분리:
- 박스 1: 한국 4채널 (메인 매출원)
- 박스 2: 해외 4채널 (글로벌 매출원)
- ※ 채널별 예약 분리는 아고다 시트가 통합 집계라 V1에서는 호텔 단위 통합 + 국가 분포로 대체
