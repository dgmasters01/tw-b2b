**상태**: 확장 중 / 시드 영상 누적 단계
**프레이밍**: "우리는 한국을 넘어 전세계로 확장하고 있습니다"

## 2. 위계 표시 원칙

### UI 표시 순서 (모든 화면)
- **한국 박스 = 위** / **해외 박스 = 아래**
- 한국 = 활성 / 해외 = 확장 중
- 한국 박스가 더 크거나, 같은 크기라도 위 정렬

### 동급 표기 금지
- ❌ "8채널에 노출됩니다" (한국·해외 동급으로 보임)
- ✅ "한국 4채널 + 해외 4채널 확장 중" (위계 명시)

### 시드 단계 약점을 강점으로 뒤집기
- ❌ "다국어 채널 시드 단계" (약점 노출)
- ✅ "전세계로 확장 중인 회사. 지금 합류 = 영상 자산 선점" (미래 가치)

## 3. 화면별 적용

### 매니저 대시보드 홈 탭
헤더 신뢰 자산 아래 메인 약속 카드:
```
한국 4채널 (메인) → 예약 N건 발생
해외 4채널 (확장) → 영상 시드 누적 중
```

### 영상 탭 채널 표
- 한국어 4채널을 위에, 다국어 4채널을 아래로 정렬
- 라벨: 한국 = "메인 진입" / 해외 = "추가 노출"

### 고객 분석 탭
채널별 분석 박스를 **2개로 분리**:
- 박스 1: 한국 4채널 합산 (즉시 매출)
- 박스 2: 해외 4채널 합산 (시드 누적)

## 4. 채널 활성화 V2 분리

YouTube Analytics API 연동 (조회수·CTR·전환률)는 V1 범위 밖.
별도 BL: **BL-CHANNEL-ANALYTICS-V2** (V1 안정화 후)

V1에서 채널별 데이터는:
- 한국 4채널: 예약 건수 + 매출 (실증)
- 해외 4채널: 게시 완료 영상 N편 + 게시일 (현재 데이터 한계)

## 5. 시드 채널 활성화 사업 전략 (별도 작업 — 킵 리스트)

다국어 채널 활성화는 V2에서 별도 사업 전략 작업.

**"성장하는 회사 = 지금 합류 = 일찍 들어간 호텔 유리"** 포지셔닝.
