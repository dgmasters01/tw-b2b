## 🔴 매니저 대시보드 (가입 후 = 운영 화면) — 영업 멘트 제거

### 정책 1: Hero 카피 줄 전체 제거
**모든 탭(홈/영상/결제/문의/고객 분석)에서 "예약 방문하게 합니다" 류 영업 멘트 사용 금지**

❌ 사용 금지:
- "우리의 콘텐츠는 당신의 호텔을 예약 방문하게 합니다"
- "지금 이 순간도 8개 채널에서 노출 중"
- "$854K 매출을 만들었습니다" 같은 자랑형 멘트

✅ 운영 정보형으로 교체:
- "Hotel Manager Dashboard" (페이지 헤더)
- "본편 영상 + 추가 노출 옵션 통합 집계" (분석 탭 부제)
- "8개 유튜브 채널에서 노출 중인 영상" (영상 탭 부제)

### 정책 2: 매니저 대시보드 탭별 카피 톤

| 탭 | 톤 | 카피 예시 |
|---|---|---|
| 🏠 홈 | 운영 현황 요약 | "이번 계약 기간 운영 현황" |
| 🎬 영상 | 사실 나열 | "한국어 4채널 · 다국어 4채널" |
| 📝 결제 | 거래 명세 | "결제 내역 · 인보이스 · 영수증 · 계약서" |
| 💬 문의 | 지원 채널 | "gohotelwinners Customer Success · 평균 4시간 이내 답변" |
| 📊 고객 분석 | 데이터 해설 | "8개 채널·6개 언어를 통해 들어온 예약 데이터입니다" |

### 정책 3: 숫자 표기는 사용 가능 — 단, 자랑형 X
- ✅ "총 예약 142건 · 확정 매출 $32,180" (운영 데이터)
- ✅ "전월 +18%" (변화율)
- ❌ "우리가 9백만 조회를 만들어 드립니다" (자랑형)
- ❌ "메인 호텔 매출 $854K 달성" (영업 멘트)

### 정책 4: 채널·언어 표기
- 매니저 대시보드에서는 채널 위계를 "운영 정보"로만 표시 (영업 자랑 X)
- ✅ "한국 4채널 + 해외 4채널 모두 운영 중"
- ❌ "당신 호텔이 전세계 9개국에 노출되고 있어요!"
