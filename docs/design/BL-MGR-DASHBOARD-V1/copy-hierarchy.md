## 카피 사용 규칙

1. **5단 위계 순서 변경 금지** — Hero → Proof → Result/Vision/Promise → Risk → Price
2. **단어 변경 금지** — "예약 방문하게 합니다", "전세계로 확장하고 있습니다" 등 확정 표현 그대로 사용
3. **한국 4채널 / 해외 4채널 위계 절대 보존** — 동급 표기 금지
4. **"마케팅 회사", "광고", "노출 도달" 같은 추상명사 금지** — 구체 명사만 사용
5. **시간 단위 (7년, 8년 등) 영업 카피에 노출 금지** — 누적 숫자만
