## 7. 탭별 핵심 결정

### 🏠 홈 탭
- Hero 카피 줄 제거 (정정 #2)
- 운영 현황 카드: 본편 영상 게시 상태, 채널별 게시 일정, 다음 결제일
- 채널 위계 v3 적용 (대만 단독)

### 🎬 영상 탭
- 본편 영상 = 카드 1개 (상단)
- 한국 4채널 위 / 해외 4채널 아래 정렬
- 추가 노출 옵션: 0편이면 섹션 숨김, 1편 이상이면 별도 섹션 표시
- 世界就是家 라벨 = "🇹🇼 대만"만 (TW·HK·CN 라벨 금지)

### 📝 결제 탭
- 결제 카드: 1차 계약 / 2차 계약 (있을 시) 분리 카드
- 카드 상단 3분할 라벨: 선결제일 / 노출 시작 / 보장 기한 종료
- 중복 한 줄 멘트 삭제 (정정 #5)
- 각 결제 카드: **인보이스 / 영수증 / 계약서** 3버튼
- 예약 명세표 7컬럼: 예약번호 / 예약일 / 체크인 / 체크아웃 / **국가** / 금액 / 상태 (정정 #6, 기기 컬럼 폐기)

### 💬 문의 탭
- 담당자: **gohotelwinners Customer Success** + 🎧 헤드셋 아이콘 아바타
- 응답 시간: 평균 4시간 이내 · 한국어, English
- **상단 FAQ 5개 접힘식 박스** (정정 #8):
  1. 영상 게시 일정은 어떻게 정해지나요?
  2. 호텔 정보(사진·설명)를 추가/변경하고 싶어요
  3. 예약은 어떻게 집계되나요? (8채널 통합?)
  4. 6개월 보장 기한 후에는 어떻게 되나요?
  5. 계약을 갱신하거나 종료하려면?
- **검색바 + 필터 5개 + 페이지네이션** (정정 #9):
  - 필터: 전체 / 답변 대기 / 답변 완료 / 내가 보낸 / 정보 변경 신청
  - 페이지당 10건
  - 6건 미만이면 자동 숨김

### 📊 고객 분석 탭
- 본편 + 추가 노출 통합 표시 (정정 #10)
- 채널 박스 2개(한국/해외) 삭제 (정정 #11)
- KPI 4카드: 총 예약 / 확정 매출 / 총 박야 / **노쇼 매출** (정정 #12)
- 노쇼 정책: **노쇼 = 매출 포함** (호텔이 객실 비워둔 정상 운영), 별도 카드, 0건일 때 회색 "없음" 정직 표시
- 취소 정책: 취소 = 매출 제외 (체크인 전 정상 환불 절차)
- 국가별 분포 차트: 4개국 (한국·일본·미국·대만)
- 월별 추이 차트 6개월
- 예약 흐름: 영상 노출 → Agoda 진입 → 예약 완료 → 체크인
- "곧 추가될 분석" 섹션: ADR / 인기 요일·시즌 / 채널별 조회 수 추이 → **V1.1 예정** 표시
