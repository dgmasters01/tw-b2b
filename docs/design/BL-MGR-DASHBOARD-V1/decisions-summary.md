## 5. 진행 상태 5단계 (이미지 015 기준)

```
1. 결제 완료 (3월 10일)
2. 정보 등록 (3월 14일)
3. 영상 제작 (5월 1일)
4. 언어 노출 진행 중 (2/6개 언어)  ← 현재 단계
5. 6개월 운영 (9월 20일까지)
```

**선결제 시나리오 분기** (이미지 013):
```
결제 완료 → 노출 대기 → 계약 진행 중 → 잔여 예약 정산 → 최종 마감
```

## 6. 회계 기준 (이미지 009 권장 A안)

- **예약 귀속**: `booked_at` 기준 (예약 버튼 누른 날 = 그 계약 기간)
- **매출 인정**: 체크아웃 완료 + 노쇼 (호텔이 돈 받음)
- **취소**: 매출 외 분리 표시
- **계약 종료 후 들어온 예약**: 어디에도 속하지 않음 (정상)

## 7. 고객 분석 7개 인사이트 (5번째 탭, 이미지 011/008 보강)

| # | 인사이트 | 라이브 컬럼 |
|---|---|---|
| 1 | 국가별 예약 분포 | `customer_country` |
| 2 | 국가별 예약 금액 | `customer_country × SUM(booking_amount_usd)` |
| 3 | 평균 박수 분포 | `nights` |
| 4 | 평일/주말 비율 | `checkin_date` 요일 |
| 5 | 인원 구성 | `num_adults + num_children` (V1 원본 표시) |
| 6 | 객실 타입 분포 | `room_type` 원본 |
| 7 | 예약 리드타임 | `booked_at → checkin_date` |

- **채널별 V1**: 예약 건수 + 매출만 (조회수·CTR은 V2 별도 BL)
- **PDF/엑셀 다운로드**
- **1차 / 2차 / 전체 기간 토글**

## 8. 영상 탭 (이미지 004/010/012)

- 본편 + TravelWinners 무상 추가 + 2차 본편 (재계약 시)
- 채널 표 7컬럼: 유튜브 채널 / 언어 / 타겟 국가 / 게시일 / 상태 / 영상 보기
- 상태 뱃지: 노출 중 / 게시 예정 / 준비 중 / 종료
- 헤더 카피: "9M회 영상 조회 — 당신 호텔도 이 흐름에 있습니다"
