# 계약서 처리 정책 (옵션 C 확정)

**BL**: BL-MGR-DASHBOARD-V1
**확정일**: 2026-05-18
**적용 범위**: sales.html 결제 직전 / 매니저 대시보드 결제 탭

---

## 1. 결정 배경

호텔 매니저 결제 시 별도 계약서 협상 절차를 두지 않고, **전자상거래법상 약관 동의로 계약 성립**시키되, 결제 후 호텔 맞춤 계약서 PDF를 자동 발행하여 회계·법무 처리 편의를 제공한다.

검토했던 3개 옵션:
- 옵션 A: 결제 전 PDF 전송 + 서명 회수 (느림, 결제 전환율 저하)
- 옵션 B: 결제만으로 계약 성립, PDF 발행 없음 (회계 처리 어려움)
- **옵션 C (확정)**: 약관 체크박스로 계약 성립 + 결제 후 PDF 자동 발행 + Resend 이메일 송부

---

## 2. 옵션 C 작동 흐름

### Stage 1: 결제 직전 (sales.html)
```
[ 결제 박스 ]
  호텔명: Seaside Hotel
  플랜: 6개월 보장 노출
  금액: $200 USD

  ☐ gohotelwinners 이용약관에 동의합니다 (전자상거래법상 계약 성립 효력)
     ▸ 약관 전문 보기 (모달)

  [ PayPal로 결제하기 ]  ← 약관 동의 체크 시에만 활성화
```

**전자상거래법 효력**: 약관 동의 체크박스 + 결제 완료 = 계약 성립 (대한민국 전자상거래 등에서의 소비자보호에 관한 법률 제13조 기반)

### Stage 2: 결제 완료 직후 (백엔드 자동)
1. PayPal webhook 수신
2. Supabase에 결제 row INSERT
3. **계약서 PDF 자동 생성** (호텔명·기간·금액·약관 본문 포함)
4. **Resend 이메일 송부**: 매니저 가입 이메일로 PDF 첨부 발송
5. Supabase Storage에 PDF 영구 보관 (매니저 대시보드 결제 탭에서 재다운로드용)

### Stage 3: 매니저 대시보드 결제 탭
각 결제 카드에 **3개 버튼** 노출:
- **인보이스** (결제 청구 명세 — 결제 전 견적용 양식)
- **영수증** (결제 완료 확인 — 회계 증빙용)
- **계약서** (옵션 C로 자동 생성된 PDF — 법무 보관용)

세 종류 모두 PDF, 모두 호텔명 맞춤, 모두 Supabase Storage에 영구 보관.

## 3. 계약서 PDF 구성

### 필수 포함 항목
1. **계약 당사자**
   - 갑: 호텔명 + 매니저 이름 + 이메일
   - 을: gohotelwinners (베트남 법인 정보)
2. **계약 기간**: 선결제일 + 노출 시작일 + 보장 종료일
3. **계약 금액**: $200 USD (또는 갱신 금액)
4. **서비스 내용**: 8개 채널·6개 언어 노출, 6개월 보장
5. **보장 조건**: 6개월 0건 시 100% 환불
6. **약관 본문**: 매니저가 체크박스로 동의한 약관 전문
7. **결제 정보**: PayPal Transaction ID + 결제 일시
8. **자동 발행 표시**: "본 계약서는 약관 동의 및 결제 완료에 따라 자동 생성된 전자 계약서입니다"

### PDF 메타데이터
- 파일명: `gohotelwinners_contract_{hotel_id}_{YYYYMMDD}.pdf`
- 작성자: gohotelwinners Operations
- 보관 위치: Supabase Storage `contracts/{hotel_id}/`

---

## 4. 재계약(갱신) 시 처리

- 기존 계약 종료 전 갱신 결제 → **새 계약서 PDF 별도 생성**
- 결제 탭에 1차 계약서·2차 계약서가 카드별로 분리되어 표시
- 같은 호텔의 N차 계약서는 시간순으로 정렬

---

## 5. 분쟁 발생 시 효력

- 약관 동의 체크박스 클릭 로그 (Supabase에 timestamp + IP 저장)
- 결제 완료 PayPal Transaction ID
- 자동 생성된 PDF의 SHA-256 해시값
- 위 3개로 전자상거래법상 계약 성립 입증

---

## 6. UI 정책

### 결제 직전 약관 체크박스
- 체크 안 했을 때 결제 버튼 비활성화 (회색 처리)
- 약관 전문 보기 = 모달로 띄움 (새 탭 X — 결제 흐름 끊김 방지)

### 결제 탭 3개 버튼
- 3개 버튼 동일 크기로 정렬 (인보이스 / 영수증 / 계약서)
- 클릭 시 새 탭에서 PDF 즉시 다운로드 (Supabase Storage 직접 링크)
- 계약서 버튼은 결제 완료된 카드에만 표시 (결제 대기 카드에는 표시 X)
