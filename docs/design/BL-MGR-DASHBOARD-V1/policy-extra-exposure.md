# 추가 노출 옵션 정책 (Extra Exposure)

**BL**: BL-MGR-DASHBOARD-V1
**확정일**: 2026-05-18
**적용 범위**: 매니저 대시보드 영상 탭 + 결제 탭 + 고객 분석 탭

---

## 1. 정의

**추가 노출 옵션** = 본편 영상 외에 별도 견적으로 추가 게시되는 영상 (호텔 주변 명소·시즌 콘텐츠·신관 오픈 영상 등).

본편 = $200 1회 결제로 제공되는 6개월 보장 노출의 기본 단위.
추가 노출 = 본편의 **서비스성 부속물** (별도 상품 아님).

---

## 2. 핵심 원칙

### 원칙 1: 0편이 기본 (Opt-in)
- 모든 신규 호텔은 **추가 노출 0편으로 시작**
- 매니저가 별도 견적 요청 → CS 승인 → 결제 → N편 추가
- **0편일 때 UI 처리**: 매니저 대시보드 영상 탭의 "추가 노출" 섹션 자체를 숨김 (빈 박스로 채우지 않음)

### 원칙 2: 본편 계약 기간 종속
- 추가 노출의 **보장 기한 = 본편 계약 종료일과 동일**
- 본편 종료일 이후 추가 노출만 별도로 연장되지 않음
- 본편 연장(갱신) 시 추가 노출도 함께 연장됨

### 원칙 3: 매출 집계 통합
- 추가 노출 영상으로 들어온 예약도 **본편 매출과 통합 집계**
- 고객 분석 탭에서 본편/추가 분리 표시 X (분석 탭 정정 #10과 일관)
- 영상 탭에서만 영상 자체는 본편/추가 시각적으로 구분

### 원칙 4: 견적 = 별도 협의
- 추가 노출 가격은 호텔별·콘텐츠별 다름 (정찰가 X)
- 매니저가 문의 탭에서 견적 요청 → CS 견적 답변 → 결제 탭에서 결제
- 결제 후 영상 탭에 "추가 노출" 섹션이 보이기 시작

## 3. UI 적용

### 영상 탭
```
0편일 때:
  본편 영상 카드만 표시
  "추가 노출" 섹션 = 숨김 (섹션 헤더도 X)

1편 이상일 때:
  본편 영상 카드 (상단)
  ── 구분선 ──
  추가 노출 영상 카드들 (하단, "추가 노출" 헤더)
```

### 결제 탭
```
본편 결제 카드: 항상 표시
추가 노출 결제 카드: 결제 1건 이상 있을 때만 표시
```

### 고객 분석 탭
- 본편 + 추가 노출 통합 매출만 표시 (분리 X)
- KPI 카드 "총 예약 건수" = 본편 + 추가 통합

### 문의 탭 FAQ 1
> Q. 영상 게시 일정은 어떻게 정해지나요?
> A. 본편은 선결제 완료일로부터 4~6주. 추가 노출은 별도 견적·결제 후 약 2~4주 내 게시.

## 4. 매니저 입장 시나리오

### 시나리오 A: 신규 가입 (0편 추가)
1. sales.html 결제 → 본편 6개월 보장
2. 매니저 대시보드 영상 탭 → 본편 영상만 표시, "추가 노출" 섹션 없음
3. 호텔 신관 오픈 시: 문의 탭에서 "신관 오픈 영상 1편 추가 견적" 문의
4. CS 견적 답변 → 결제 → 영상 탭에 "추가 노출" 섹션 등장

### 시나리오 B: 본편 갱신 시 추가 노출 자동 연장
1. 본편 갱신 결제 → 본편 보장 기한 6개월 연장
2. 기존 추가 노출 영상의 보장 기한도 자동 연장 (본편과 동일)

### 시나리오 C: 본편 종료 후 추가 노출 단독 X
1. 본편 보장 기한 종료
2. 추가 노출도 동시 종료
3. 추가 노출만 단독 연장은 불가 (본편 갱신 필수)
